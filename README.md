# Admin Form Backend

This is the backend of the Admin Form built using **Node.js**, **Express**, and **MongoDB**.

##  Features

- Admin Registration and Login
- JWT Authentication
- Protected Routes
- MCQ Upload
- Student Entry Form

##  Technologies Used

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT
- bcryptjs
- dotenv

git clone https://github.com/Sufyan898/adminForm.git
